# Clip Collide!

Project for [GreatUniHack 2025](https://greatunihack.com).

## Setup

1. Clone the repo

```sh
git clone https://github.com/fieryhenry/guh-2025
```

2. Install dependencies

```sh
pip install -r requirements.txt
```

3. Run

```sh
flask --app src/main.py run
```

Add the `--debug` to enable hot reloading.

It may take a while to run on first use to download the model used for genre detection
